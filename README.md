# MarketShield AI

A production-ready full-stack marketplace fraud analyzer that inspects e-commerce listings, compares prices, and classifies fraud risk using heuristics plus Gemini AI.

## Quick Start
```bash
npm install
cp .env.example backend/.env
npm run dev
```

## Project Structure
- `frontend/` – React + Vite client
- `backend/` – Express API server
- `docs/` – setup, API, and deployment documentation
- `deploy/` – production Nginx reverse proxy sample

See `docs/README.md` for the full documentation.
